# Checklist de Mudança Programada - Engenharia de ISP

## 📋 Sobre o Projeto

Sistema web para gerenciamento e documentação de mudanças programadas em redes de provedores de internet (ISP). Desenvolvido para padronizar o processo de registro de alterações em equipamentos de rede, seguindo as melhores práticas de engenharia.

## 🚀 Funcionalidades

- **Preenchimento automático** de data, hora e número da mudança
- **Campos editáveis** para todas as informações do checklist
- **Tabela dinâmica** de equipamentos com adição/remoção de linhas
- **Upload de arquivos** com pré-visualização de imagens
- **Campos condicionais** que aparecem conforme necessidade
- **Geração de relatório** com atualização automática do horário de término
- **Impressão** e exportação para PDF

## 📂 Estrutura do Checklist

1. Identificação da Mudança
2. Objetivo da Mudança
3. Equipamentos Envolvidos
4. Análise de Impacto
5. Pré-Check (Backup, Firmware, Compatibilidade, Recursos, Saúde)
6. Plano de Execução
7. Plano de Rollback
8. Execução
9. Validação Pós-Mudança
10. Monitoramento
11. Evidências
12. Encerramento
13. Aprovações

## 🔧 Personalização

### Alterar a logo:
- Substitua a URL da imagem no `src` da tag `<img>` no cabeçalho

### Alterar campos padrão:
- Modifique os valores dos placeholders nos inputs
- Ajuste os textos das labels conforme necessidade

### Adicionar novas opções:
- Para checkboxes: adicione novos `<label>` com `<input type="checkbox">`
- Para radios: adicione novas opções no grupo correspondente
